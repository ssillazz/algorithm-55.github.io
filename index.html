Algorithm 55

Develop an algorithm that reads N any integers, where N is provided by the keyboard, and inform which of those N integers are different from 55. For example, if the number entered is 12, the program should read 12 integers and print on the screen which are different from 55.

Tradução: Elabore um algoritmo que leia N números inteiros quaisquer, onde N é fornecido pelo teclado, e informe quais desses N números inteiros são diferentes de 55. Por exemplo, se o numero digitado for 12, o programa deve ler 12 valores inteiros e imprimir na tela quais são diferentes de 55.
